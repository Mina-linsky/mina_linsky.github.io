Welcome to My Static Website!

This is a basic example of a static HTML page. It's a file that is served to a user's browser exactly as it is stored on the server.

What is HTML?

HTML, or HyperText Markup Language, is the standard language for creating web pages. It defines the structure of a page using a series of tags.

What is a Static Site?

A static website is made up of a fixed set of files (HTML, CSS, JavaScript, images) that are delivered to the user without any changes. Unlike dynamic websites, there's no server-side processing to generate the content on the fly.

A Quick List of Things

  • HTML is the skeleton.
  • CSS is the style.
  • JavaScript adds interactivity.
